WHAT IS PROGRAMMING ??

  • Computer programming is a way of giving computers instructions in the form of code about what they should do next.
  • A programming language provides a way for a programmer to express a task so that it could be understood and executed by a computer.

WHY PROGRAMMING ??

  • The backbone of Technology Companies like Google, Facebook, Microsoft, Apple, Amazon, and many others.

  • It is fun as using Programming, we can create our own games, personal blogs, social networking sites or an e-commerce platform.
