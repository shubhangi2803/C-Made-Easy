 OPERATOR   DESCRIPTION  OPERATION    EXAMPLE 
+ To add two values Addition

a+b

(5+2=7)
- To subtract one value from another Subtraction

a-b

(5-2=3)
* To multiply two values Multiplication

a*b

(5*2=10)
/ To divide one number by another Division

a/b

(5/2=2.5)
% To find the remainder after division Modulus

a%b

(5%2=1)
++ To increase the value by 1 Increment

a++

(a=5, a++ =6)
-- To decrease the value by 1 Decrement

a--

(a=5, a-- =4)
